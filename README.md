# 🏠 Ev Tahmini Projesi

Bu proje, makine öğrenmesi teknikleri kullanılarak ev fiyatlarının tahmin edilmesini amaçlayan bir Python uygulamasıdır. Kullanıcıdan alınan ev özelliklerine göre model, yaklaşık fiyat tahmini yapar.

---

## 🚀 Proje Amacı

Gerçek veri setleri üzerinde çalışarak:
- Veri analizi süreçlerini öğrenmek
- Makine öğrenmesi regresyon modellerini uygulamak
- Ev fiyat tahmini yapan bir sistem geliştirmek

---

## 🧠 Kullanılan Teknolojiler

- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Proje Özellikleri

- Veri seti yükleme ve temizleme
- Eksik verilerin işlenmesi
- Özellik mühendisliği (feature engineering)
- Regresyon modeli ile eğitim
- Ev fiyatı tahmini
- Model performans analizi
